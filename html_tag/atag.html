HTML <a>tag


The <a> tag ពាក្យពេញ (anchor tag) ក្នុងភាសា HTML ប្រើដើម្បីបង្កើតជាតំណភ្ជាប់ ដែលយើងហៅថា hyperlink នៅលើទំព័រ webpage, វាអាចធ្វើការភ្ជាប់ពី ទំព័រមួយទៅកាន់ទំព័រមួយផ្សេងទៀត, ភ្ជាប់ទៅកាន់ទីតាំងផ្សេងៗនៅលើទំព័រតែមួយ ឬទីតាំងណាមួយ នៅលើទំព័រផ្សេងក៏បាន។



ចំណាំ៖ រាល់ការប្រើ <a> tag ដើម្បីបង្កើតតំណភ្ជាប់ ទៅកាន់កន្លែងណាមួយ ចំបាច់ត្រូវមាន href attribute, ប្រសិនបើយើងមិនប្រើ href attribute ជាមួយនោះទេ វានិងមិនអាចតភ្ជាប់ទៅកាន់ទីតាំងណាមួយបានឡើយ។

<a> tag with hreflang attribute


hreflang attribute ជាការកំណត់ ភាសារនៅលើ Link element របស់យើង, ដើម្បីបញ្ជាក់ប្រាប់ទៅ browser នៅភាសារដែលយើងកំពុងប្រើ ដូច្នេះម៉ាសុីន ស្វែងរកនឹងផ្តល់ភាពងាយ និងស្វែងរករហ័ស សម្រាប់ search user.

<!DOCTYPE html>
<html lang="en">
<body>
<p>page using khmer language</a>
<a href="../Word/Lesson1.html" hreflang="kh" target="_blank">Microsoft Word / Lesson1</a>
</body>
</html>
