HTML Canvas Tutorial



The HTML canvas> element ប្រើសម្រាប់គូសជា graphics on a web page.

The graphic នៅខាងលើ បង្កើតចេញជាមួយនិង canvas>.

វាបង្ហាញចេញជាបួន elements: a red rectangle, a gradient rectangle, a multicolor rectangle, and a multicolor text.

អ្វីជា HTML Canvas?

The HTML <canvas> element ប្រសម្រាប់គូសជា graphics, ដើម្បីអោយរូបមានចលនា, តាមរយះ JavaScript.

The <canvas> element ផ្ទុកសម្រាប់តែ graphics តែប៉ុណ្ណោះ. យើងត្រូវប្រើ JavaScript ដើម្បីជាជំនួយក្នុងការគូស graphics.
Canvas គូសចេញជារៀងមួយចំនួនដូចជា paths, boxes, circles, text, ហើយនឹងការបន្ថែមជា images.
