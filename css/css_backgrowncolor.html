CSS Background Color

Background-Color property កំណតជាផ្ទៃពណ៌ទៅលើ element

ជាឧទាហរណ៏

ដើម្បីធ្វើការកំណត់ផ្ទៃពណ៌ ទៅលើទំព័រគឺយើងត្រូវធ្វើតាមរបៀបនេះ

body {
      background-color: pink
       }

មើលកាសាកល្បងពេញ

ជាមួយ​ និង CSS, Color ការប្រើប្រាស់ ច្រើនកំណត់ដូចខាងក្រោម៖
⦿ a valid color name - like "red"
⦿ a HEX value - like "#ff0000"
⦿ an RGB value - like "rgb(255,0,0)"

កំណត់ផ្ទៃពណ៌លើ html element

របៀបកំណត់ background color ទៅលើ html element ផ្សេងៗគ្នា​
<h1>, <p>, <div>

h1 {
      background-color: red
       }
p {
      background-color: #ff0000
       }
div {
      background-color: rgb(255,0,0)
       }

មើលកាសាកល្បងពេញ

Opacity / Transparency

The ​​​Opacity មុខងារកំណត់កម្រិតស្រាលរបស់ពណ៌ ការកំណត់តំលៃ ចាប់ពី 0.0 ដល់ 1.0

opacity: 1

opacity: 0.6

opacity: 0.3

opacity: 0.1



ជាឧទាហរណ៏

ដើម្បីធ្វើការកំណត់ផ្ទៃពណ៌ ទៅលើទំព័រនិងកំណត់កម្រិត opacity ត្រឹម 0.5

body {
      background-color: pink
      opacity: 0.5
       }


ចំណាំ៖ នៅពេលដែលយើងធ្វើការកំណត់ Opacity ដើម្បីបានកម្រិតពណ៌ ស្រាល ទៅលើ background elementណាមួយ នោះ រាល់ child element ទាំងអស់ និងមានការប្រែលប្រួលដូចគ្នា ដែលអាចអោយការអានមានការពិបាក
