HTML class Attributes



និយមន័យ និង​កាប្រើប្រាស់

The class attribute កំនត់បញ្ជាក់ មួយឬច្រើ class names សម្រាប់ element.

The class attribute ប្រើច្រើនសម្រាប់ជាចំណុចសំដៅកាន់ class នៅក្នុង​ style sheet, ម្យាងវិញទៀង, វាក៏អាចប្រើតាមរយះ javacript ផងដែរ(via the HTML DOM) ដើម្បីធ្វើការផ្លាស់ប្តូរ html element ជាមួយនិងភាពជាក់លាក់នៃclass.

ជាឧទាហរណ៏

Use the class attribute in html.

<!DOCTYPE html>
<html>
<head>

    <style>
      h1.intro{
       color: blue
       }
      p.important{
       color: green
       }
    </style>
</head>
<body>
      <h1 class="intro" >Header 1</h1>
      <p>A paragraph</p>
      <p class="important">Note that this is an important paragraph</p>
<html>
<body>

Out put

Header 1

A paragraph

Note that this is an important paragraph


Browser Support

Attribute
accesskey Yes Yes Yes Yes Yes

Syntax

      <element class="classname">


Attribute Value

Value Description
classname Specifies one or more class names for an element. To specify multiple classes, separate the class names with a space, e.g. . This allows you to combine several CSS classes for one HTML element.

Naming rules:

▪ Must begin with a letter A-Z or a-z
▪ Can be followed by: letters (A-Za-z), digits (0-9), hyphens ("-"), and underscores ("_")

ឧទាហរណ៏ បន្ថែម

Add multiple classes to one HTML element:

<!DOCTYPE html>
<html>
<head>

    <style>
      h1.intro1{
       color: blue;
       text-align: center
       }
      .important1{
       background-color: yellow
       }
    </style>
</head>
<body>
      <h1 class="intro1 important1" >Header 1</h1>
      <p>A paragraph</p>
<html>
<body>

Out put

Header 1

A paragraph

Using Javacript to add a yellow background color to the first element with class="example" (index 0).

<!DOCTYPE html>
<html>
<body>

      <div class="example" >Fist div element with class="example".</div>
      <div class="example" >Second div element with class="example".</div>
      <p>Click the botton to add a yellow background color to the first div element woth class="example" (index 0).</p>
      <button onclick="myFunction()">Try it</button>

<script>
function myFunction() {
var x = document.getElementsByClassName("example");
x[0].style.backgroundColor = "yellow" }
</script>
<html>
<body>

Out put

Fist div element with class="example".
Second div element with class="example".

Click the botton to add a yellow background color to the first div element woth class="example" (index 0).
