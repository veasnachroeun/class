HTML Canvas Graphics



The HTML <canvas> element ប្រើសម្រាប់គូសជា graphics នៅលើ web page.

The graphic នៅខាងឆ្វេងគឺបានបង្កើតឡើងដោយប្រើ <canvas>. វាបានបង្ហសញ 4 elements: a red rectangle, a gradient rectangle, a multicolor rectangle, and a multicolor text.

អ្វីជា HTML Canvas?

The HTML <canvas> element ប្រសម្រាប់គូសជា graphics, ដើម្បីអោយរូបមានចលនា, តាមរយះ JavaScript.

The <canvas> element ផ្ទុកសម្រាប់តែ graphics តែប៉ុណ្ណោះ. យើងត្រូវប្រើ JavaScript ដើម្បីជាជំនួយក្នុងការគូស graphics.
Canvas គូសចេញជារៀងមួយចំនួនដូចជា paths, boxes, circles, text, ហើយនឹងការបន្ថែមជា images.

Canvas Examples



A canvas ជារាងចតុកោណមួយនៅលើ HTML page. ទ្រង់ទ្រាយដើមរបស់ canvas គ្មានបន្ទាត់ហើយក៏គ្មានផ្ទុកអ្វីនៅខាងក្នុងដែរ.
លក្ខណះសម្គាល់របស់វាដូចខាងក្រោមនេះ:

<canvas id="mycanvas" width="200" height="200"><canvas>

Note: ត្រូវកំណត់ id attribute ជានិច្ច (ដើម្បីភ្ជាប់ជាមួយ script), ហើយ width និង height attribute ដើម្បីកំណត់ទំហំរបស់ canvas. ដើម្បីបន្ថែម border, ត្រូវប្រើ style attribute.

ខាក្រោមជាឧទាហរណ៏ដំបូងក្នុងការប្រើ, empty canvas:
