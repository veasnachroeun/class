CSS Syntax (រូបមន្ត)

h1 {color:blue; font-size:32px}


◉ The selector ជាចំណុចផ្តោតទៅកាន់ html element ដែលយើងត្រូវធ្វើការ រចនាទៅកាន់។
◉ The declaration block មាន declaration មួយ ឬច្រើន ដែលវាត្រូវបានបែងចែក ដោយសញ្ញាក្បៀស (;) សម្រាប់ការប្រកាសមួយៗ (declaration)។
◉​ declaration មួយៗ មាន css property name ហើយនិង value ដែលខាន់ចែកដោយសញ្ញា ចុចពីរ (:)។
◉ css declaration ចាប់ពីពីរឡើងទៅត្រូវខាន់ចែកដោយសញ្ញាក្បៀស (;) និង declaration block ត្រូវបើក និងបិទ ដោយសញ្ញាខ្នៀបពីរ{}

ជាឧទាហរណ៏

P {
color: red;
text-align: center;
}

មើលកាសាកល្បងពេញ

ពន្យល់បន្ថែមលើឧទាហរណ៏ខាងលើ


◉ P គឺជា css selector (ដែលវាជាគោលដៅមួយ ដែលយើងធ្វើការកំណត់ ទៅកាន់ html element ដើម្បីធ្វើការរចនាទៅលើ <p>)។
◉ color គឺជា property និង red គឺជា value ។
◉ text-align គឺជា property និង center គឺជា value ។
