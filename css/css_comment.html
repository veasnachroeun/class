អ្វីទៅជា CSS Comment?

CSS Comment មិនត្រូវបានបង្ហាញចេញនៅលើ browser នោះទេ តែវាជាកំណត់ត្រាមួយដែលជាជំនួយក្នុងការសរសេរកូដ របស់យើង ក្នុងការចំណាំដើម្បីងាយស្រួលសម្រាប់កែប្រែកូដ នៅពេលក្រោយ
     ទីតាំងប្រើរបស់ CSS Comment គឺនៅចន្លោះ <style> ដែលផ្តើមដោយ សញ្ញា ខាន់ចែកនិងផ្កាយ /* និងបិទបញ្ចប់វិញ ដោយសញ្ញា ផ្កាយ និងខាន់ចែក */
