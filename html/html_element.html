HTML Elements


- HTML Elements ជាកំណត់សម្គាល់ ដោយ ការ បើក tag, បន្ទាប់មក ជាបណ្តុំ ឯកសារ, និងការ បិត tag.​

បើក tag បណ្តុំ ឯកសារ content បិទ tag
<p> This is Paragraph content </p>
<h1> This is heading content </h1>
<div> This is division content </div>
<br> none none


- ដូច្នេះ <p>.....</p> ហៅថា html element, <h1>.....</h1> ជា html element មួយផ្សេងទៀត
ហើយក៏មាន html element មួយចំនួន គឺមិនតម្រូវអោយមានការ បិទ tag​ នោះទេ ដូចជា <br>, <hr> <img...> ទាំងនេះត្រូវបានហៅថា​ empty elements ឬ void elements

Nested HTML Elements


គឺជាបណ្តុំ html element មួយ ដែលក្តោបយក រាល់ element ច្រើនផ្សេងទៀ ហៅថា (nested html element)
-​ នៅក្នុង ឧទាហរណ៏ខាងក្រោម មានផ្ទុកនៅ html element ចំនួន បូន (<html>, <body>, <h1>, <p>).

ឧទាហរណ៏៖

<!DOCTYPE html>
<html>
<body>
      <h1>This is Heading</h1>
      <p>This is Paragraph</p>
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ

This is Heading

This is Paragraph


បកស្រាយ បន្ថែម៖

- The <html> element គឺជា element ដំបូងគេង ដែលក្តោបយក រាល់ element ទាំងអស់
- វាគឺចាប់ផ្តើមឡើងដោយ <html> និង បិតបញ្ចប់វិញ ដោយ </html>
-ដូចនេះ នៅខាងក្នុង <html> element គឺមាន <body> element

<body>
      <h1>This is Heading</h1>
      <p>This is Paragraph</p>
</body>


- The <body> element គឺជា element មួយ ដែលក្តោបយក element ផ្សេងទៀតផងដែរ
- វាគឺចាប់ផ្តើមឡើងដោយ <body> និង បិតបញ្ចប់វិញ ដោយ </body>
-ដូចនេះ នៅខាងក្នុង <body> element គឺមាន <h1> element និង <p> element

      <h1>This is Heading</h1>
      <p>This is Paragraph</p>


- The <h1> element គឺជា element មួយ ដែលសម្គាល់ជា heading
- វាគឺចាប់ផ្តើមឡើងដោយ <h1> និង បិតបញ្ចប់វិញ ដោយ </h1>

      <h1>This is Heading</h1>


- The <p> element គឺជា element មួយ ដែលសម្គាល់ជា Paragraph
- វាគឺចាប់ផ្តើមឡើងដោយ <p> និង បិតបញ្ចប់វិញ ដោយ </p>

      <p>This is Paragraph</p>


Never Skip the end tag

HTML elements មួយចំនួនបង្ហាញចេញយ៉ាងត្រឹមត្រូវ បើទោះជាយើងភ្លេច បិទវាក៏ដោយ តែជាទំលាប់ល្អ យើងគួរតែកុំភ្លេចបិទវាអោយសោះ

ឧទាហរណ៏៖

<!DOCTYPE html>
<html lang="en">
<body>
      <h1>This is Heading No close end tag
      <p>This is Paragraph No close end tag
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ

This is Heading No close end tag

This is Paragraph No close end tag

HTML elements ដែលគ្មានបណ្តំ ឯកសា ហៅថា empty elements.



The <br> tag កំណត់មុខងារដើម្បីចុះបន្ទាត់ វាជា empty element ដែលមិនមានការបិទ tag:
