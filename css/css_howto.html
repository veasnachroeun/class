វិធីក្នុងការបញ្ចូល css

រាល់ពេល browser អាន style sheet វានិងធ្វើការ format html document តាមលំដាប់ អ្វីដែលមានក្នុង style sheet


មានបីវិធីក្នុងការបញ្ចូល css ទៅក្នុង html

ដើម្បីទាញយក មុខងារ css មកប្រើប្រាស់ ក្នុង html បាន យើងអាចធ្វើតាមវិធីសាស្ត្រ បីយ៉ាងដូចខាងក្រោមនេះ
     ◉ External CSS
     ◉ Internal CSS
     ◉ Inline CSS



External CSS

ជាមួយនឹងការប្រើប្រាស់ External CSS យើងអាចធ្វើការកែប្រែទ្រង់ទ្រាយទាំងអស់បាននៅលើវេបសាយរបស់យើង ដោយគ្រាន់តែធ្វើការកែលើ file តែមួយប៉ុណ្ណោះ។
    - គ្រប់ html page មួយៗ ត្រូវតែបញ្ចូលតំណភ្ជាប់ <link> css នៅចន្លោះ <head> tag របស់ html

Syntax
     <link rel="stylesheet" href="name.css">

ជាឧទាហរណ៏

External styles ត្រូវបានសំគាល់ដោយ <link> element នៅផ្នែកខាងក្នុង <head> របស់ html

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="stylesheet.css">
</head>
<body>
    <h1>H1 is heading</h1>
    <p>h2 is smaller than h1</p>
    <p>This is paragraph</p>

</body>
</html>

មើលកាសាកល្បងពេញ

External styles
ត្រូវបានសរសេរទុកក្នង file css ដែលយើងអាចធ្វើការកែប្រែបានគ្រប់ពេល
The External .css file មិនត្រូវមាន html tag ណាមួយនោះទេ
ខាងក្រោមនេះគឺជា code css ដែលទុកក្នុង "stylesheet.css" file

body {
background-color: gray;
}
h1 {
color: blue;
text-align: center;
}
p {
color: red;
magin-left: 25px;
}

ចំណាំ៖
សម្រាប់តំលៃ property 20px ត្រូវសរសេរជាប់គ្នាមិនអាចដកឃ្លាបានទេ
- ត្រឹមត្រូវ ៖ margin-left: 20px
- មិនត្រឹមត្រូវ ៖ margin-left: 20 px


Internal CSS

Internal styles
    ការប្រើប្រាស់បានតែលើ html page មួយប៉ុណ្ណោះ ពីព្រោះរាល់ code ទាំងអស់ដែលប្រើជាលក្ខណះ Internal css ត្រូវបានផ្ទុកនៅលើ ផេកមួយណាដែលយើងចង់ធ្វើការរចនា ហើយការកែប្រែណាមួយ ក៏បានតែផេកដែលប្រើ ជាលក្ខណះ Internal ដែរ វាខុសពី ការប្រើជាលក្ខណះ External ដែរយើងអាច ធ្វើការកែប្រែច្រើនផេកបាន ក្នុងពេលតែមួយ។
    ចំពោះការប្រើ Internal style ត្រូវបានសំគាល់ ដោយប្រើ <style> នៅចន្លោះ​ <head> tag element នៅលើ html page ផ្ទាល់

ជាឧទាហរណ៏

Internal styles សំគាល់ដោយ <style> element នៅចន្លោះផ្នែក <head> របស់ html

<!DOCTYPE html>
<html>
<head>
<style>
body {
background-color: darkorchid;
}
h1 {
color: darkgreen;
margin-left: 45px;
}
</style>
</head>
<body>
    <h1>H1 is heading</h1>
    <p>h2 is smaller than h1</p>
    <p>This is paragraph</p>

</body>
</html>

មើលកាសាកល្បងពេញ



Inline CSS

Inline styles
    ប្រើសម្រាប់រចនា ទៅលើ tag element របស់ html ណាមួយដែលយើងចង់តាក់តែង ដោយវាមិនប៉ះពាល់ទៅលើ tag ផ្សេង។
    ​ដើម្បីប្រើ Inline styles បាន គឺយើងត្រូវប្រើ <style> attribute ទៅលើ html tag element ណាមួយដែលចង់បាន ដោយភ្ជាប់ជាមួយ property css

ជាឧទាហរណ៏

Inline styles សំគាល់ដោយ <style> attribute របស់ html tag element

<!DOCTYPE html>
<html>
<head>
</head>
<body>
    <h1 style="color:red; text-align:center;">H1 is heading</h1>
    <p style="color:blue; margin-left:55px;">h2 is smaller than h1</p>
    <p>This is paragraph</p>

</body>
</html>

មើលកាសាកល្បងពេញ

ការនែនាំ(Tip)៖

     ចំពោះការប្រើប្រាស់លើមុខងារ Inline styles វាមិនផ្តល់លក្ខណះងាយស្រួលច្រើនដល់ការរចនានៅលើផេករបស់យើងនោះទេ (វាមានភាពច្របូកច្របល់ច្រើន) វិធីល្អយើងគួរយកវាមកប្រើតិចតួចបានហើយ។


Multiple Style Sheets

     ប្រសិនបើមានមុខងារ css ដូចគ្នាច្រើន​ ដែលត្រូវបានយើងយកមក រចនា លើ hml element ផ្សេងៗគ្នា នោះតំលៃចុងក្រោយរបស់ property Css និងត្រូវបានកំណត់ក្នុងការប្រើ

ជាឧទាហរណ៏
យើងនឹងធ្វើការរចនា ទៅលើ html tag element តែមួយ ដោយប្រើមុខងារពីរ គឺ Internal style និង inline style ដោយកំណត់យក property css ដូចគ្នា និងតំលៃខុសគ្នា (លិទ្ឋផល browser នឹងធ្វើការអានពីលើចុះមកក្រោម និងកំណត់យកចំនុចចុងក្រោយបំផុតក្នុងការប្រើ)

<!DOCTYPE html>
<html>
<head>
<style>
p {
color: red;
margin-left: 45px;
}
</style>
</head>
<body>
    <p style="color:blue; font-size:35px;">browser និងកំណត់យកតំលៃចុងក្រោយ គឺ ពណ៌ខៀវ និង ទំហំអក្សរ 35px</p>
</body>
</html>
