អ្វីទៅជា CSS Selectors?

CSS Selectors ជាចំនុចគោលមួយរបស់​ html element ដែលយើងផ្តោតទៅកាន់ ដើម្បីធ្វើការរចនាទៅលើវា។


យើងអាចធ្វើការបែងចែក css Selectors ជាប្រាំដំណាក់ផ្សេងៗគ្នា

◉ Simple selectors (ផ្តោតទៅ element លើ name, id, class)
◉ Combinator selectors (ផ្តោតទៅ element ដែលមានទំនាក់ទំនងគ្នា)
◉ Pseudo-class selectors (ផ្តោតទៅ element ដែលបានរៀបរាប់)
◉ Pseudo-elements selectors (ផ្តោតទៅ element ទៅតាមផ្នែកដែលបានកំណត់)
◉ Attribute selectors (ផ្តោតទៅ attribute រឺ attribute value)

ស្វែងយល់បន្ត

The css element selectors

element selector សំដៅទៅ​ html element ផ្តោតយក tag name element

ជាឧទាហរណ៏

រាល់ tag <p> element ទាំងអស់ និងត្រូវ តម្រៀប នៅចំនុចកណ្តាល នៃ page និង អក្សរពណ៌ ក្រហម

P {
color: red;
text-align: center;
}

មើលកាសាកល្បងពេញ

The css id selectors

id selector សំដៅទៅ​ id attribute របស់ html element ដោយសម្គាល់យកចំណុចជាក់លាក់មួយ ក្នុងចំណោម tag element ផ្សេងៗ
ដើម្បីកំណត់យក​ tag element ដែលមាន id attribute យើងប្រើសញ្ញាទ្រុងជ្រូគ (#) ពីមុខតូអក្សរ(ឈ្មោះid)

ជាឧទាហរណ៏

រាល់ tag element ទាំងអស់ ដែលមាន id attribute ឈ្នោះ test1 និងត្រូវ តម្រៀប នៅចំនុចកណ្តាល នៃ page និង អក្សរពណ៌ ក្រហម លើកលែងតែ tag ណាដែលគ្មាន id=test1

#test1 {
color: red;
text-align: center;
}

មើលកាសាកល្បងពេញ

ចំណាំ៖ មិនអាចប្រើ លេខនៅពីមុន ឈ្មោះ id បានទេ


The css class selectors

class selector សំដៅទៅ​ class attribute របស់ html element ដោយសម្គាល់យកចំណុចជាក់លាក់មួយ ក្នុងចំណោម tag element ផ្សេងៗ
ដើម្បីកំណត់យក​ tag element ដែលមាន class attribute យើងប្រើសញ្ញាចុច (.) ពីមុខតូអក្សរ(ឈ្មោះclass)

ជាឧទាហរណ៏

រាល់ tag element ទាំងអស់ ដែលមាន id attribute ឈ្នោះ testclass និងត្រូវ តម្រៀប នៅចំនុចកណ្តាល នៃ page និង អក្សរពណ៌ ក្រហម លើកលែងតែ tag ណាដែលគ្មាន class=testclass

.testclass {
color: red;
text-align: center;
}

មើលកាសាកល្បងពេញ

យើងក៏អាចប្រើ tag html element ដើម្បីកំណត់យក​ class name ណាមួយបានផងដែរ មើល ឧទាហរណ៏ខាក្រោម

ជាឧទាហរណ៏

p.testclass1 {
color: red;
text-align: center;
}

មើលកាសាកល្បងពេញ

យើងក៏អាចធ្វើ ការ select ទៅលើ class ច្រើនជាងមួយបានផងដែរ

ជាឧទាហរណ៏

ខាងក្រោមនេះយើងនឹងធ្វើការ select ទៅលើ <p> tag និង class="color", class="font"

p.colorcenter{
color: red;
text-align: center;
}
p.font{
font-size:: 250%;
}

មើលកាសាកល្បងពេញ

ចំណាំ៖ មិនអាចប្រើ លេខនៅពីមុន ឈ្មោះ class បានទេ

The CSS Universal Selectors

The universal selector ប្រើសញ្ញាផ្កាយ(*) ដើម្បីធ្វើការ selects យក HTML elements​ ទាំងអស់ដែលមាននៅលើ page.

ជាឧទាហរណ៏

css code ខាងក្រោមនេះ គឺនឹងធ្វើការ ទៅលើ html element ទាំងអស់

* {
color: red;
text-align: center;
}

មើលកាសាកល្បងពេញ


The CSS Grouping Selectors

The grouping selector ប្រើជាក្រុមសម្រាប់ធ្វើការរចនាដើម្បីអោយចេញជាផលដូចគ្នា

ជាឧទាហរណ៏

ក្រឡេកមើល css code ខាងក្រោមនេះ (ដែលយើងធ្វើការរចនាទៅលើ h1, h2, p element ដែលមាន style ដូចគ្នា)

h1 {
color: red;
text-align: center;
h2 {
color: red;
text-align: center;
p {
color: red;
text-align: center;
}

វានឹងកាន់តែមានភាពងាយស្រួលនឹងចំណេញពេលក្នុងការសរសេរ code​ប្រសិនយើងចងវាជាគ្រុប ដោយបែងចែក element មួយៗ ដោយសញ្ញា (,) មើលឧទាហរណ៏ខាងក្រោម៖

h1, h2, p {
color: red;
text-align: center;
}

មើលកាសាកល្បងពេញ
