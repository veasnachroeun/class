The formenctype Attribute

The input formenctype attribute មស្រាប់កំណត់របៀប form-data ដែលត្រូវបំឡែង នៅពេល submitted (សម្រាប់តែ forms ជាមួយ method="post" តែមួយប៉ុណ្ណោះ).

The formenctype attribute ដំណើរការជាមួយនឹងប្រភេទ input មានដូចជា : submit and image.

Note: This attribute overrides the enctype attribute of the <form> element.

</form>

The formmethod Attribute

The input formmethod attribute កំណត់បង្ហាញ the HTTP method សម្រាប់ form-data ទៅកាន់ action URL.

Note: This attribute overrides the method attribute of the <form> element.

The formmethod attribute ដំណើរការជាមួយនិងប្រភេទ input ដូចជា: submit and image.

The form-data can be sent as URL variables (method="get") or as an HTTP post transaction (method="post").

Notes on the "get" method:

This method appends the form-data to the URL in name/value pairs
This method is useful for form submissions where a user want to bookmark the result
There is a limit to how much data you can place in a URL (varies between browsers), therefore, you cannot be sure that all of the form-data will be correctly transferred
Never use the "get" method to pass sensitive information! (password or other sensitive information will be visible in the browser's address bar)

Notes on the "post" method:

This method sends the form-data as an HTTP post transaction
Form submissions with the "post" method cannot be bookmarked
The "post" method is more robust and secure than "get", and "post" does not have size limitations

<form​ action="action_test.asp" method="get" target="_blank"​>
     <label for="fname">First name:</label><br>
     <input type="text" id="fname" name="fname" value="veasna"​ disabled><br>
     <label for="lname">Last name:</label><br>
     <input type="text" id="lname" name="lname" value="Chroeun"><br>
     <input type="submit" value="Submit using Get"><br>
     <input type="submit" method="post" value="Submit using Post"><br>
</form>

The formnovalidate Attribute

The input formnovalidate attribute សម្រាប់ធ្វើការកំណត់ទៅលើ <input> element ដែលមិនមានសុពលភាពនៅពេល submitted.

Note: This attribute overrides the novalidate attribute of the <form> element. The formnovalidate attribute ដំណើរការជាមួយប្រភេទ input: submit.

ជាឧទាហរណ៏៖
ការកំណត់ max date, a min date, ហើយនិងទៅតាមជួនៃតំលៃ:

<form​ action="action_test.asp">
     <label for="email">Email address:</label><br>
     <input type="email" id="email" name="email" required><br>
     <input type="submit" value="Submit"><br>
     <input type="submit" formnovalidate ="formnovalidate" formtarget="_blank" value="Submit with novalidate"><br>
</form>

The novalidate Attribute

The novalidate attribute ជា <form> attribute.

នៅពេលដែលវត្តមានរបស់, novalidate កំណត់លើ form-data និន្ន័យគ្មានសុពលភាពទាំងអស់នឹង បញ្ជូន.

ជាឧទាហរណ៏៖

<form​ action="action_test.asp" novalidate>
     <label for="email">Email address:</label><br>
     <input type="email" id="email" name="email" required><br>
     <input type="submit" value="Submit"><br>
