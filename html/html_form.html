The HTML <form> element ផ្ទុកនៅប្រភេទ input elements ផ្សេងៗគ្នា មានដូចជា text fields, checkboxes, radio button, submit buttons, etc.

The <input> Element

The HTML <input> element ប្រើច្រើនបំផុតនៅលើ form element.

An <input> element ត្រូវបានយកមកប្រើតាមលក្ខណះច្រើនយ៉ាង, គឺវាអាស្រ័យទៅលើ type attribute

ខាងក្រោមនេះជា ឧទាហរណ៏ មួយចំនួន៖

Type Description
<input type="text"> Displays a single-line text input field
<input type="checkbox"> Displays a checkbox (for selecting zero or more of many choices)
<input type="submit"> Displays a submit button (for submitting the form)
<input type="button"> Displays a clickable button

គ្រប់ប្រភេទ input elements ទាំងអស់ត្រូវបានរៀបរៀងទុកក្នុងមេរៀន HTML Input Types

The <label> Element

The <label> tag ប្រើច្រើនក្នុង form elements.

The <label> element ជាប្រយោជន៏ដល់អ្នកប្រើ ដើម្បីកំណត់សម្គាល់ រាល់ចំនុចមួយៗ

The for attribute of the <label> ត្រូវដូច ទៅនិង The id attribute of the <input> element ដើម្បីចងពួកវាជាប់គ្នា

Radio Buttons

The <input type="radio"> សម្គាល់ ជា radio button

Radio button អាចអោយយើងធ្វើការ select ម្តងបាន មួយ

ជាឧទាហរណ៏៖

     <input type="radio" id="css" name="b_yourclass" value="CSS">
     <label for="css">CSS</label><br>
     <input type="radio" id="java" name="b_yourclass" value="JAVA">

The Submit Button

The <input type="submit"> សម្គាល់ ជា Button សម្រាប់បញ្ជូន form data ទៅកាន់ form-handler.

form-handler ជាប្រភេទfile មួយដែលមាននៅលើ server សម្រាប់ដំណើរការ input data.

The form-handler មានបញ្ជាក់ក្នុង form's action attribute.
