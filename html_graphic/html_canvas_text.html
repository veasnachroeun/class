Drawing Text on the Canvas

ដើម្បីគូសចេញជា text នៅលើ canvas, មុខងារ property និង methods សំខាន់បំផុតគឺ:

1- font - កំណត់សំគាល់ជា font properties សម្រាប់ text
2-​ fillText(text,x,y) - គូស "filled" text នៅលើ canvas
3-​ strokeText(text,x,y) - គូស text នៅលើ canvas (no fill)


Using fillText()

<!DOCTYPE html>
<html>
<body>
     <canvas id="fillText" width="250" height="100" style="border: 1px solid blue">Your Browser doesn't suppot with canvas tag</canvas>
<script>
var canvas = document.getElementById("fillText");
var ctx = canvas.getContext("2d");

ctx.font = "30px, Arial";
ctx.fillText("Your Class KH",10,50,);
</script>
</html>
