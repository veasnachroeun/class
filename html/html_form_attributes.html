HTML Form Attribute



ក្នុងមេរៀនគឺយើងមកសិក្សាបន្ថែមលើ លក្ខណះរបស់ attributes លើ <form> element

The Action Attribute


The Action Attribute កំណត់សកម្មភាពដើម្បីដំណើរការពេល ដែល form ត្រូវបានបញ្ជូន

ជាទូទៅ the form data ត្រូវបានផ្ញើទៅកាន់ file នៅលើ server ពេលដែល user ចុចលើ submit button.

ឧទាហរណ៏ ខាងក្រោមនេះ នឹងបង្ហាញពីរបៀប form data ផ្ញើទៅកាន់ file មួយ ឈ្មោះ "test_page.php" (អនុវត្តន៏ជាក់ស្តែនៅក្នុងមេរៀន PHP)

ជាឧទាហរណ៏៖

<form action="/test_page.php">
     <label for="fname">First name:</label><br>
     <input type="text" id="fname" name="fname"><br>
     <label for="lname">Last name:</label><br>
     <input type="text" id="lname" name="fname"><br><br>
     <input type="submit" value="submit"><br>
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ







គន្លឹះ ប្រសិនបើលុបចេញ action attribute, ដំណើការ form និង កំណត់លើ current page.

The Taget Attribute


The target attribute សម្រាប់កំណត់យកទីតាំង បង្ហាញចេញពេលដែល form ត្រូវបានបញ្ជូនចេញ

The target attribute អាចមានមួយក្នុចំណោម តំលៃ target ដូចខាងក្រោម

Value Description
_blank The response is displayed in a new window or tab
_self The response is displayed in the current window
_parent The response is displayed in the parent frame
_top The response is displayed in the full body of the window
framename The response is displayed in a named iframe

ជាឧទាហរណ៏៖

<form​ action="/action_page.php" target="_blank" >

ចំណាំ៖ ក្នុងលក្ខខណ្ឌ មិនកំណត់យក តំលៃ របស់ taget ណាមួយ តំលៃដើមរបស់វាគឺ _self


The Method Attribute

The method​ កំណត់បញ្ជាក់វិធីជាក់លាក់ នៃ http​ method នៅពេល form ចាប់ផ្តើមដំណើការ ហើយវាក៏មានពីរប្រភេទផងដែរ ក្នុងការកំណត់បង្ហាញនៃ http method នោះគឺ GET និង​ POST. ដូចនេះ the method attribute តំលៃរបស់វាមានពីរ គឺ get និង​ post ដែលយើងយកមកប្រើនៅក្នុង form element

Attribure Value:

👉 GET: ជាទម្រង់តម្លៃដើម, នៅក្នុង get method, បន្ទាប់ពីធ្វើការបញ្ជូន form, តម្លៃក្នុងform នឹងធ្វើការបង្ហាញ ពត៌មាន address bar នៅលើ New browser tab, វាមានទំហំកំណត់ អាចផ្ទុកបានប្រហែល 3000 តួអក្សរ, ប្រើ get សម្រាប់រាល់ DATA ដែលមិនមានសុវត្ថិភាពប៉ុន្នោះ គឺមិនសម្រាប់ពត៌មានណាដែលសំខាន់នោះទេ។

👉 POST: នៅក្នុង​ post method, បន្ទាប់ពីធ្វើការបញ្ជូន form, តម្លៃ របស់ form មិនបង្ហាញចេញ address bar ក្នុង tab ថ្មី ដូចទៅនឹង get method នោះទេ, វាបន្ថែមទម្រង់ទិន្ន័យទៅក្នុង តួរបស់ http request. ទំហំ size មិនកំណត់, វាមិន suppot ជាមួយ Bookmark ទេ។

ក្នុងឧទាហរណ៏នេះ យើងប្រើ get method ពេលដែលប្រើបញ្ជូន form data

<form​ action="/action_page.php" method="get" >

ក្នុងឧទាហរណ៏នេះ យើងប្រើ post method ពេលដែលប្រើបញ្ជូន form data

<form​ action="/action_page.php" method="post" >

គន្លឺះ៖ ត្រូវប្រើ post ប្រសិនបើ DATA របស់យើងផ្ទុក ឯកសារសំខាន់ ឬ ពត៌មានផ្ទាល់ខ្លួន

The Autocomplete Attribute

The autocomplete attribute ប្រើដើម្បីកំណត់បង្ហាញភាពច្បាស់លាស់មួយទៅលើ autocomplete ដោយកំនត់បិទ ឬបើក (on ឬ off). នៅពេលដែល autocomplete attribute ត្រូវបានកំណត់ ជា on, browser និងធ្វើការបំពេញដោយស្វ័យប្រវត្តិ រាល់ពត៌មានរបស់ user ដែលធ្លាប់បានបញ្ចូលពីមុនមក។ វាដំណើរជាមួយនិង input field ជាច្រើន ដូចជា text, search, URL, email, password date pickers, range, and color.

ជាឧទាហរណ៏៖

A form ជាមួយនិង autocomplete on:

<form​ action="/action_page.php" autocomplete="on" >

The Novalidate Attribute

The novalidate attribute នៅលើ Form tag គឺដើម្បីបិទរាល់ validation ទាំងអស់ក្នុង input elements នៅពេលដែលធ្វើការបញ្ជូន form.

ខាងក្រោមនេះ ជាឧទាហរណ៏ ពីរ ក្នុងការកំណត់ novalidate និង មិនកំណត់ novalidate

ឧទាហរណ៏(ទី១)៖

<form​ action="/action_page.php" novalidate>
     <label for="fname">Fist Name </label>
     <input type="text" id="fname" placeholder="first name" required><br>
     <label for="lname">Last Name</label>
     <input type="text" id="lname" placeholder="Last Name" required><br>
     <input type="submit" value="Submit">
</form>

List of all <form> Attributes

Attribute Description
accept-charset Specifies the character encodings used for form submission
action Specifies where to send the form-data when a form is submitted
autocomplete Specifies whether a form should have autocomplete on or off
enctype Specifies how the form-data should be encoded when submitting it to the server (only for method="post")
method Specifies the HTTP method to use when sending form-data
name Specifies the name of the form
novalidate Specifies that the form should not be validated when submitted
rel Specifies the relationship between a linked resource and the current document
target Specifies where to display the response that is received after submitting the form
