HSL Color Value

    HSL ប្រើជាតំណាងអោយ Hue, Saturation, និង lightness.
នៅក្នុងភាសារ css របៀបប្រើប្រាស់ color លក្ខណះ hue, saturation, lightness (hsl) ជាទម្រង់ដូចនេះ
hsl(hue, saturation, lightness)

នេះជាតំលៃនិមួយៗក្នុងការកំណត់
- HUE តំលៃរង្វិលជុំរបស់វា ចាប់ពី 0 ដល់ 360, 0 គឺ red, 120 green និង 240 blue
- Saturation កំណត់តំលៃជាភាគរយ, 0% ស្មើរ shade of gray, 100% ស្មើរ full color
- lightness ក៏កំណត់ជា %ដែរ, 0% ស្មើរ black, 100% ស្មើរ white, 50% light or dark

hsl(0, 100%, 50%)

hsl(120, 100%, 50%)

hsl(240, 100%, 50%)

hsl(0, 100%, 100%)

Saturation

ជាមុខងារសម្រាប់ធ្វើការបន្ថែមឬបន្ថយពណ៌តាមកម្រិត, កំណត់ 100% ជាកម្រិតពេញ របស់ពណ៏​ 50% ជាកម្រិតពាក់កណ្តាលពណ៌ប្រផេះ កម្រិតនេះយើងអាចមើលឃើញពណ៌ដើម 0% គឺកម្រិតប្រផេះសុតតែម្តង

hsl(0%, 100%, 50%)

hsl(0, 50%, 50%)

hsl(0, 0%, 50%)

Lightness

ជាកម្រិតពន្លឺរបស់ពណ៌ ការកំណត់ពន្លឺទៅលើពណ៌ទៅតាមការកំណត់ដូចនេះ ចំណុច 0% កំណត់យកពន្លឺងងឹត (ខ្មៅ) ចំណុច 50% ជា 50% នៃពន្លឺភ្លឺ និង ងងឹត ចំណែកកម្រិ 100% គឺពេញពន្លឺ (សរ)។
