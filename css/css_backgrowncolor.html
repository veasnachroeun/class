CSS Background Color

Background-Color property កំណតជាផ្ទៃពណ៌ទៅលើ element

ជាឧទាហរណ៏

ដើម្បីធ្វើការកំណត់ផ្ទៃពណ៌ ទៅលើទំព័រគឺយើងត្រូវធ្វើតាមរបៀបនេះ

body {
      background-color: grey
       }

មើលកាសាកល្បងពេញ

ជាមួយ​ និង CSS, Color ការប្រើប្រាស់ ច្រើនកំណត់ដូចខាងក្រោម៖
⦿ a valid color name - like "red"
⦿ a HEX value - like "#ff0000"
⦿ an RGB value - like "rgb(255,0,0)"

កំណត់ផ្ទៃពណ៌លើ html element

របៀបកំណត់ background color ទៅលើ html element ផ្សេងៗគ្នា​
<h1>, <p>, <div>

h1 {
      background-color: red
       }
p {
      background-color: #ff0000
       }
div {
      background-color: rgb(255,0,0)
       }

មើលកាសាកល្បងពេញ

HSLA Value

hsla color value ជាធាតុផ្សំបន្ថែមមួយទៀត ទៅលើ hsl ដោយភ្ជាប់ជាមួយនិង alpha ដែលសម្រាប់កំណត់បន្ថែមទៅលើលក្ខណះរបស់ពណ៌
ទម្រង់ របស់ hsla គឺដូចនេះ
hsla(hue, saturation, lightness, alpha)
ចំពោះតំលៃ​ កំណត់របស់ alpha គឺរវាង 0.0 ដល់ 1.0

hsla(10, 100%, 75%, 0)

hsla(10, 100%, 75%, 0.2)

hsla(10, 100%, 75%, 0.4)

hsla(10, 100%, 75%, 0.6)

hsl(10, 100%, 75%, 0.8)

hsla(10, 100%, 75%, 1)
