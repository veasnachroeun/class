CSS Introduction

CSS គឺជាភាសារមួយដែលយកមកប្រើដើម្បីធ្វើការរចនាទៅលើវេបសាយ


ស្វែងយល់បន្ថែមទៅលើតួនាទីផ្សេងរបស់ CSS

∎ CSS មកពីពាក្យ “Cascading Style Sheets”
∎ CSS សម្រាប់ផ្តល់​ លក្ខណះទ្រង់ទ្រាយ ទៅអោយ html element ក្នុងការបង្ហាញ ចេញនៅផ្ទៃខាងក្រៅ ទៅតាមប្រភេទ អេក្រង់ផ្សេងគ្នា, និងប្រភេទ media ផ្សេងៗគ្នាផងដែរ។
∎ CSS ជួយសម្រួលការងាររបស់យើងបានជាច្រើន វាអាយគ្រប់ផេកជាច្រើនអោយដំណើរការក្នុងពេលតែមួយ។
∎ ចំពោះ External StyleSheets គឺត្រូវបានផ្ទុកនៅលើ CSS Files
