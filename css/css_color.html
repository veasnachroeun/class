<h1 style="background-color: gray"> element មួយនេះត្រូវបានធ្វើការកំណត់ជា background ពណ៌ប្រផេះ </h1>​
<h2 style="background-color: orange"> element មួយនេះត្រូវបានធ្វើការកំណត់ជា background ពណ៌ទឹកក្រូច </h2>​
<p style="background-color: red"> element មួយនេះត្រូវបានធ្វើការកំណត់ជា background ពណ៌ក្រហម </p>

Color Value

color values

យើងអាចធ្វើការកំណត់តំលៃ (value) របស់ color property ជា​ RGB, HEX, HSL, RGBA, HSLA មួយណាក៏បាន

      ខាងក្រោមនេះយើងនឹង ធ្វើការកំណត់ background-color ទៅលើ <div> បី ដោយប្រើ value ជា RGB, HEX និង HSL

RGB(255, 195, 0)

#FFC300

HSL(46, 100%, 50%)

      សម្រាប់ <div> ពីរ ខាងក្រោមនេះ យើងនឹងធ្វើការកំណត់ ជាមួយនឹង RGBA, HSLA ដោយកំរិតព្រាល (Alpha) យើងយក 50%

RGBA(255, 195, 0, 0.5)

HSLA(46, 100%, 50%, 0.5)

ជាឧទាហរណ៏

<p style="background-color:#FFC300")>...</p>
<p style="background-color:hsl(46, 100%, 50%")>...</p>

<p style="background-color:rgba(255, 195, 0, 0.5")>...</p>
<p style="background-color:hsla(46, 100%, 50%, 0.5")>...</p>
