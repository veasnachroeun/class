HTML Basic Examples


- នៅក្នុង មេរៀននេះយើងនឹង ធ្វើការ បង្ហាញ ជា ឧទាហរណ៏ មូលដ្ឋានគ្រឹះ មួយចំនួន របស់ html.​
- សូមអ្នកកុំមានការបារម្មណ៏ ប្រសិនបើយ យើង ធ្វើការប្រើ tag ណាមួយដែលអ្នកមិនទាន់បានសិក្សារ​ នៅឡើយ។

HTML Documents

- រាល់ ឯកសារ HTML ទាំងអស់ ចាំបាច់ត្រូវផ្តើមឡើង ដោយការប្រកាស <!DOCTYPE html>.
-​ HTML ខ្លួនរបស់វាផ្ទាល់ ផ្តើមជាមួយ​ <html> និងបញ្ចប់ដោយ <html>.​
-ការបង្ហាញជាផ្នែកមួយនៃ ឯកសាររបស់ HTML គឺរវា <body> និង </body>.

ជាឧទាហរណ៏៖

<!DOCTYPE html>
<html>
<body>

<h1>This is the head tag (h1)</h1>
<p>This is the paragraph tag (p)</p>

</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ

This is the head tag (h1)

This is the paragraph tag (p)


The <!DOCTYPE> Declaration


-​ ការប្រកាស​ជា <!DOCTYPE> គឺតំណាងអោយជាប្រភេទ ឯកសារ, និងផ្តល់ភាពងាយស្រួលអោយ browsers ក្នុងការបង្ហាញ web pages អោយបានត្រឹមត្រូវ។
-​ វាមានតែមួយប៉ុណ្ណោះ នៅផ្នែកខាងលើបង្អស់នៃទំព័រ (នៅផ្នែកខាងលើ HTML tags).
- ការប្រកាស​ជា <!DOCTYPE> គឺមិនប្រកាន់អក្សរ ធំឬតូចនោះទេ.
-​ ការប្រកាស​ជា <!DOCTYPE> សម្រាប់ HTML5 គឺ:

<!DOCTYPE html>

HTML Headings

- HTML headings យកមកប្រើ ដើម្បីសរសេរ ជាលក្ខណះចំណងជើងដោយមានជាអក្សរឌិតធំ គឺកំណត់ជាមួយ និង <h1> ដល់ <h6> tags (head tag).
- <h1> កំនត់ជា heading ដែលសំខាន់បំផុតមួយ. <h6> កំនត់ជា heading ចុងក្រោយបង្អស់:

ឧទាហរណ៏៖
សរសេរនៅ h1 tag ដល់ h6 tag

<!DOCTYPE html>
<html>
<body>
      <h1>This is heading, write under the h1 tag</h1>
      <h1>This is heading, write under the h2 tag</h2>
      <h1>This is heading, write under the h3 tag</h3>
      <h1>This is heading, write under the h4 tag</h4>
      <h1>This is heading, write under the h5 tag</h5>
      <h1>This is heading, write under the h6 tag</h6>
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ

HTML Paragraphs

- HTML paragraphs​ យកមកប្រើ ដើម្បីសរសេរ ជាលក្ខណះអក្សរទូទៅធម្មតា គឺកំណត់ជា <p> tags (paragraph tag).

ឧទាហរណ៏៖

<!DOCTYPE html>
<html>
<body>
      <p>This is Paragraph, write under the p tag<p>
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ

This is Paragraph, write under the p tag


HTML Links

- HTML links គឺសម្រាប់បង្កើតតំណភ្ជាប់ ទៅកាន់ទីតាំងណាមួយ ដោយកំណត់ប្រើ <a> tags (anchor tag).

ឧទាហរណ៏៖

<!DOCTYPE html>
<html>
<body>
      <a href="https://www.youtube.com/">Go to YOUTUBE</a>
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ

Go to YOUTUBE

កំណត់ចំណាំ៖​
- តំណភ្ជាប់ link ទៅកាន់ទីតាំងណាមួយ គឹត្រូវតែ បញ្ជាក់បន្ថែម ដោយប្រើ href attribute (Hypertext REFerence)
- Attributes គឺត្រូវបានប្រើសំរាប់ ផ្តល់ជាពត៌មានបន្ថែមទៀត អំពី HTML elements.
- យើងនឹងលំអិតអំពី attributes នៅមេរៀនក្រោយ



HTML Images

HTML images are defined with the tag. - HTML images ប្រើដើម្បីទាញយករូបភាព វាដំណើរការទៅបាន ជាមួយ​ <img> tags (image tag).
- ដោយរួមបញ្ចូលជាមួយនិង attribute មួយចំនួន ដូចជា៖
      + source file (src) attribute ប្រើសំរាប់ទាញយករូបភាពពីប្រភពណាមួយ
      + alternative text (alt) attribute ប្រើសំរាប់កំណត់ឈ្មោះរូបភាព
      + width, និង height (width, និង height) attribute ប្រើសំរាប់កំណត់ទំហំទទឹង និង កំពស់ ទៅលើរូបភាព

ឧទាហរណ៏៖

<!DOCTYPE html>
<html>
<body>
      <img src="../Image/Your Class.jpg" alt="y-c logo" width="100px" height="100px">
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ
