HTML Uniform Resource Locators



A URL ជាពាក្យមួយផ្សេងសម្រាប់ប្រើជា Web address.

A URL ជាមាសភាពរបស់ពាក្យ (ឧ. yourclass.com) ឬ an internet protocol(IP) address (e.g. 192.168.0.1).

ការចូលទៅកាន់វេបសាយភាគច្រើនគឺប្រើឈ្មោះ, ពីព្រោះឈ្មោះគឺងាយចងចាំជាងលេខ។

URL-Uniform Resource Locator

Web browsers ទាញយក ផេក ពី Web servers ដោយប្រើ URL

A Uniform Resource Locator (URL) ប្រើជា address a document (or other data) នៅលើវេបសាយ។

Web address ដូចជា https://www.yourclass.social មើលរូបមន្តប្រើខាងក្រោម៖


ជាឧទាហរណ៏៖

scheme://prefix.domain:port/path/filename

ស្វែងយល់៖

scheme - សំគាល់ប្រភេទ អ៊ិនធឺណេត (type of internet service) និយមប្រើ គឺ http or https

prefix សំគាល់ domain prefix (ទម្រង់របស់ http គឺ www)

domain សំគាល់ឈ្មោះ domain លើ internet (ដូចជា youclass.social)

port សំគាល់ port number នៅលើ host (ទម្រង់របស់ http គឺ 80)

path សំគាល់​ path លើ server

filename សំគាល់ឈ្មោះឯកសារ

URL Schemes ប្រើទូទៅ

តារាងខាងក្រោមជា scheme ប្រើទូទៅ

Scheme Short for Used for
https Secure HyperText Transfer Protocol Secure web pages. Encrypted
ftp File Transfer Protocol Downloading or uploading files
file A file on your computer
http HyperText Transfer Protocol Common web pages. Not encrypted

URL Encoding

URLs អាច sent តាមរយះ អ៊ិនធឺណេត ដោយប្រើ ASCII character. ប្រសិនបើ URL មានផ្ទុកសំណុំអក្សរក្រៅពី ASCII, URL ត្រូវ convert ជាមុន។

URL encoding ធ្វើការ convert រាល់តួអក្សរដែលមិនមែនជា ASCII ទៅជា ASCII ដើម្បីអាចដំណើរការតាម internet បាន។

URL encoding ជំនួស រាល់តួអក្សរដែលមិនមែនជា ASCII ជាមួយ និង និមិត្តសញ្ញាភាគរយ "%" តាមរយះ hexadecimal digits.

URLs មិនមានការដកឃ្លាទេ, URL encoding និងធ្វើការជំនួសដោយ សញ្ញាបូក (+) ឬ %20 ក្នុងលក្ខ័ន្ឋមានកាដកឃ្លា។
