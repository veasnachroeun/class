CSS Border

    The CSS Border Property ប្រើដើម្បីកំណត់បន្ទាត់ ជុំវិញអត្ថបទ

    The Border Property ខ្ញុំប្រើវាដើម្បីកំណត់បន្ទាត់ ជុំវិញអត្ថបទរបស់ខ្ញុំ

    The Border-bottom Property ខ្ញុំប្រើវាដើម្បីកំណត់បន្ទាត់ ពណ៌ក្រហមនៅខាងក្រោមអត្ថបទរបស់ខ្ញុំ

    The Border-radius Property ខ្ញុំប្រើវាដើម្បីកំណត់បន្ទាត់ រាងពងក្រពើជុំវិញអត្ថបទរបស់ខ្ញុំ

    The Border-left Property ខ្ញុំប្រើវាដើម្បីកំណត់បន្ទាត់ ខាងឆ្វេងនៃអត្ថបទរបស់ខ្ញុំ

CSS Border Style

       The border-style គឺជា property មួយដែលកំណត់បែបរចនានៃព្រំ (border)។ វាមានជម្រើសនានាដូចខាងក្រោម:

* dotted កំណត់ស៊ុមបន្ទាត់មានចំណុច ចុចតៗគ្នា
* dashed កំណត់ស៊ុមមានស្នាមបន្ទាត់ជាកំណាត់ៗ បន្តបន្ទាប់
* solid កំណត់ស៊ុមមានបន្ទាត់ជាប់តែមួយ
* double កំណត់ស៊ុមមានបន្ទាត់ទ្វេដង
* groove កំណត់ស៊ុម 3D ប្រភេទជ្រាប (grooved)។ ប្រសិទ្ធិភាពអាស្រ័យលើតម្លៃពណ៌ស៊ុម (border-color)
* ridge កំណត់ស៊ុម 3D ប្រភេទកំពូង (ridged)។ ប្រសិទ្ធិភាពអាស្រ័យលើតម្លៃពណ៌ស៊ុម (border-color)
* inset កំណត់ស៊ុម 3D ប្រភេទចូលក្នុង (inset)។ ប្រសិទ្ធិភាពអាស្រ័យលើតម្លៃពណ៌ស៊ុម (border-color)
* outset កំណត់ស៊ុម 3D ប្រភេទលេចចេញ (outset)។ ប្រសិទ្ធិភាពអាស្រ័យលើតម្លៃពណ៌ស៊ុម (border-color)
* none កំណត់មិនមានស៊ុម
* hidden កំណត់ស៊ុមលាក់ (hidden)

       The border-style property អាចមានតម្លៃពីមួយដល់បួន (for the top border, right border, bottom border, and the left border).

ឧទាហរណ៏

p.dotted {border-style: dotted;}

     A dotted border.

p.dashed {border-style: dashed;}

     A dashed border.

p.solid {border-style: solid;}

     A solid border.

p.double {border-style: double;}

     A double border.

p.groove {border-style: groove;}

     កំណត់ស៊ុម 3D ប្រភេទជ្រាប (grooved)

p.ridge {border-style: ridge;}

     កំណត់ស៊ុម 3D ប្រភេទកំពូង (ridged)

p.inset {border-style: inset;}

     កំណត់ស៊ុម 3D ប្រភេទចូលក្នុង (inset)

p.outset {border-style: outset;}

     កំណត់ស៊ុម 3D ប្រភេទលេចចេញ (outset)

p.none {border-style: none;}

     កំណត់មិនមានស៊ុម

p.hidden {border-style: hidden;}

     កំណត់ស៊ុមលាក់ (hidden)

p.mix {border-style: dotted dashed solid double;}

     4 style បញ្ចូលគ្នា
