HTML Canvas Coordinates



The HTML canvas គឺជាក្រឡាចត្រង្គពីរវិមាត្រ.

នៅជ្រុងខាងឆ្វេងផ្នែកខាងលើរបស់ canvas មានកូអរដោនេស្មើរនឹង (0,0)

នៅក្នុងមេរៀនមុនគឺយើងបានប្រើប្រាស់មុខងារ: fillRect(0,0,150,75).

មានន័យថាការចាប់ផ្តើមពីជ្រុងខាងឆ្វេង (0,0) ហើយគូសជាចតុកោណ​ស្មើរ 150x75 pixels.

Coordinates Example
គំនូសចតុកោណ ដែលមានកូអរដោនេ x,y


Draw a Line

គូសបន្ទាត់ត្រង់នៅលើ Convas ដោយប្រើមុខងារដូចខាងក្រោម:

moveTo(x,y) - កំណត់ចំនុចចាប់ផ្តើមរបស់បន្ទាត់

lineTo(x,y) - កំណត់ចំនុចចុងក្រោយរបស់បន្ទាត់

ដើម្បីគូសចេញជាបន្ទាត់យើងត្រូវប្រើមុខងារមួយក្នុងចំណោមមុខងារ "ink" ទាំងអស់, ដូចជា stroke().

<!DOCTYPE html>
<html>
<body>
     <canvas id="DrawLine" width="200" height="100" style="border: 1px solid blue">Your Browser doesn't suppot with canvas tag</canvas>
<script>
var canvas = document.getElementById("DrawLine");
var ctx = canvas.getContext("2d");
ctx.moveTo (0,0);
ctx.lineTo( (200,100));
ctx.stroke();
</script>
</html>
</body>


Draw a Circle

ដើម្បីគូសអោយចេញជារង្វង់នៅលើ canvas, ត្រូវប្រើមុខងារដូចខាងក្រោមនេះ:

beginPath() - ចំនុចចាប់ផ្តើមនៃការគូស

arc(x,y,r,startangle,endangle) - បង្កើតជារង្វង់.

បង្កើតចេញជារង្វង់ជាមួយ arc(): កំណត់ start angle ទៅជា 0 និង end angle ទៅជា 2*Math.PI. តំលៃ x និង y, សម្រាប់កំណត់ទីតាំងរបស់ រង្វង់នៅត្រង់ចំនុចណាមួយនៅ លើ canvas. r កំណត់រង្វាស់កាំរង្វង់.

<!DOCTYPE html>
<html>
<body>
     <canvas id="circle" width="300" height="250" style="border: 1px solid black">Your Browser doesn't suppot with canvas tag</canvas>
<script>
var canvas = document.getElementById("circle");
var ctx = canvas.getContext("2d");
ctx.beginPath();
ctx.arc(150,125,100,0,2*Math.PI);
ctx.stroke();
</script>
</html>
</body>
