HTML Canvas Reference



The <canvas> element សម្រាប់កំណត់ទីតាំងគូសនៅក្នុង HTML page.​
The Canvas API អនុញ្ញាតអោយ JavaScript គូសចេញជារូបរាងនៅលើ canvas.​
The Canvas API អាចគូសចេញជា shapes, lines, curves, boxes, text, and images, with colors, rotations, transparencies, and other pixel manipulations.

យើងអាចបញ្ចូល canvas element នៅទីតាំងណាក៏បាន ក្នុង HTML page ជាមួយនិង <canvas> tag:

ជាឧទាហរណ៏៖

<!DOCTYPE html>
<html>
<body>
     <canvas id="canvas" width="300" height="150" style="border: 1px solid blue; margin-left: 150px;"></canvas>
</body>
</html>

កូដខាងលើបង្ហាញចេញដូចនេះ

យើងអាចធ្វើការក្នុង <canvas> element បាន ដោយរួមជាមួយនិង​HTML DOM method getElementById().
ដើម្បីគូសនៅក្នុង canvas ហើយបន្ទាប់មកយើងបង្កើត 2D context object

var myCanvas = document.getElementById("canvas");
var ctx = myCanvas.getContext("2d");

Note:
The HTML <canvas> element​ ខ្លួនរបស់វាផ្ទាល់គឺមិនអាចគូសចេញជារូបរាងផ្សេងៗបាននោះទេ.
គឺត្រូវការមុខងារ JavaScript ដើម្បីគូសចេញជារូបរាងណាមួយ
The getContext() ប្រៀបដូចជា ឧបករណ៏ដែលអាចធ្វើអោយយើងគូសនៅក្នុង Canvas បាន។


Drawing on the Canvas


បន្ទាប់ពីយើងបានបង្កើត​ 2D context, យើងអាចធ្វើការគូសនៅ canvas​ បាន
មុខងារ fillRect() method ដើម្បិគូសជាចតុកោណពណ៏ខ្មៅ ជាមួយនឹងការកំណត់ គម្លៀតកំពស់ជ្រុងខាងឆ្វេង​ 20px, និង គម្លៀតពីលើចុះក្រោម 20px, ទំហំចតុកោណនៅក្នុង Canvas គឺ 150 pixel ទទឹង និង 100 pixels កំពស់.

<body>

<script>
var myCanvas = document.getElementById("Canvas");
var ctx = myCanvas.getContext("2d");
ctx.fillRect(20,20,150,100);
</script>
</html>
</body>

Paths


វិធីទូទៅ ដើម្បីធ្វើការគូស នៅលើ ​canvas:

1- Begin a Path - beginPath()
2- Move to a Point - moveTo()
3- Draw in the Path - lineTo()
4- Draw the Path - stroke()

ជាឧទាហរណ៏៖

<!DOCTYPE html>
<html>
<body>
     <canvas id="Canvas2" width="300" height="150" style="border: 1px solid blue">Your Browser doesn't suppot with canvas tag</canvas>
<script>
var myCanvas2 = document.getElementById("Canvas2");
var ctx2 = myCanvas2.getContext("2d");
ctx2.beginPath();
ctx2.moveTo(40,40);
ctx2.lineTo(40,120);
ctx2.lineTo(150,120);
ctx2.lineTo(40,40);
ctx2.strokeStyle = "red";
ctx2.stroke();
</script>
</html>
</body>

Complete Canvas API Reference

This reference covers all Properties and Methods of the getContext("2d") Object, used to draw text, lines, boxes, circles, pictures, and more on the canvas.


Drawing Methods

មានបីវិធីដើម្បីគូសផ្ទាល់នៅលើ canvas:



Method Description
fillRect() Draws a "filled" rectangle
strokeRect() Draws a rectangle (with no fill)
clearRect() Clears specified pixels within a rectangle

Path Methods

មានបីវិធីដើម្បីគូសផ្ទាល់នៅលើ canvas:



Method Description
beginPath() Begins a new path or resets the current path
closePath() Adds a line to the path from the current point to the start
isPointInPaht() Returns true if the specified point is in the current path
moveTo() Moves the path to a point in the canvas (without drawing)
lineTo() Moves the path to a point in the canvas (without drawing)
fill() Fills the current path
rect() Adds a rectangle to the path

Colors, Styles, and Shadows



Properties Description
fillStyle Sets or returns the color, gradient, or pattern used to fill the drawing
strokeStyle Sets or returns the color, gradient, or pattern used for strokes
shadowColor Sets or returns the color to use for shadows
shadowBlur Sets or returns the blur level for shadows
shadowOffsetX Sets or returns the horizontal distance of the shadow from the shape
shadowOffsetY Sets or returns the vertical distance of the shadow from the shape
