HTML Attribute


- អ្នកបានស្គាល់រួចមកហើយ HTML tags ដែលមានដូចជា heading tags <h1>...,<h2> paragraph tag, <p> និង tag ផ្សេងមួយចំនួនទៀត, ដែលយើងយកមកប្រើវា ដើម្បីបង្កើត ទម្រង់ផ្សេងៗ ទៅតាមមុខងាររបស់វា, បន្ថែមពីលើនេះ ទៀត គឺយើងមាន html attribute ប្រើភ្ធាប់ជាមួយវា ដើម្បីផ្តល់រូបរាងបន្ថែមទៅលើ tags មួយៗ

    👉 An attribute ប្រើដើម្បី ផ្កល់រូបរាង លក្ខណះបន្ថែម ទៅលើ html element, ដែលជាទូទៅ វានគឺ ថិតនៅជាប់ និង ការចាប់ផ្តើម បើនៃ tags និមួយៗ , ដែលកំណត់ ដោយ name និង value
          ⨀ The name ជា Property ដែលយើងត្រូវជ្រើសយកមកកំណត់ នៅអ្វីដែលយើងចង់បាន, ជាឧទាហរណ៏, យើងមាន the paragraph <p> element មួយ ដែល យើងកំណត់ attribute ដោយ ជ្រើស name align ដើម្បីធ្វើការតម្រឹម អត្ថបទ តាមដែលយើងចង់បាន នៅលើ Page​
          ⨀ The value ជាអ្វីដែលយើងកំណត់ នៅលើ Property ដើម្បីអោយវាមានការផ្លាស់ប្តូរទៅតាមតំលៃដែលយើងបានកំណត់, តំលៃ (value) របស់ align Property គឺមាន បី left, center, right ជាទូទៅ យើង ដាក់វានៅក្នុង សញ្ញា ធ្មេញកណ្តុរ align="left"

ឧទាហរណ៏៖

<!DOCTYPE html>
<html lang="en">
<body>
<p align="left">This is left align</p>
<p align="center">This is center align</p>
<p align="right">This is right align</p>
</body>
</html>

The href Attribute

the href attribute សម្រាប់កំណត់ឈ្មោះ(web address)ដើម្បីបង្កើតតំណភ្ជាប់រឆ្ពស់ទៅកាន់។ ជាទូទៅយើងប្រើ href attribute ជា Property របស់ <a> tag

ឧទាហរណ៏៖
ប្រសិនបើយើងចង់បង្កើតតំណភ្ជាប់ (link) ទៅកាន់ Youtube Page ត្រូវបើក <a> tag ដោយភ្ជាប់ជាមួយនិង href attribute

<!DOCTYPE html>
<html lang="en">
<body>
<a href="https://www.youtube.com/">Click this link to the Youtube Page</a>
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ
Click this link to the Youtube Page



The src Attribute

the src attribute ប្រ់ើសម្រាប់កំណត់យកទីតាំងប្រភពរបស់រូបភាព ដែលវាជា attribute ដើរតូជា property របស់ img tag
<img> tag មុខងារសម្រាប់បង្គប់រូបភាពក្នុង html

ឧទាហរណ៏៖

<!DOCTYPE html>
<html lang="en">
<body>
<img src="/y-c_image.jpg">
</body>
</html>


      - កូដខាងលើបង្ហាញចេញ ដូចនេះ



* មានពីរវិធីក្នុងការកំណត់យកប្រភពរូបភាព
1. Absolute URL - ទាញយករូបភាពពីរប្រភពវេបសាយខាងក្រៅមកដាក់លើវេបសាយរបស់យើង
(EX. src="//http://www.domain.com/y-c_image.jpg")
2. Relative URL - ទាញយករូបភាពពីរប្រភពវេបសាយ របស់យើងផ្ទាល់ ដោយមិនប្រើ domain name គឺប្រើត្រឹមតែ slash (/y-c_image)
(EX. src="/y-c_image")



The width and height Attributes

<img> tag ក៏តម្រូវអោយមាន width និង height attribute ប្រើរួមជាមួយផងដែរ
width attribute កំណត់ទំហំ ទទឹងអោយទៅរូបភាព​ height attribute កំណត់ទំហំ បាតទៅកំពស់រូបភាព (ដោយគិតជា pixels)

ឧទាហរណ៏៖

<!DOCTYPE html>
<html lang="en">
<body>
<img src="/y-c_image.jpg"​​ width="350px" height="150px">
</body>
</html>

The alt Attribute

ការប្រើ alt attribute សម្រាប់ <img> tag
ដើម្បីអោយអ្នកប្រើកំណត់សម្គាល់រូបភាព នៅពេលដែលរូបភាពមិនត្រូវបានបង្ហាញនៅ ផេក ក្នុងលក្ខ័នណាមួយ វានិងបង្ហាញចេញជាងក្សរជំនួស

ឧទាហរណ៏៖

<html lang="en">
<body>

</body>
</html>

យើងនិងធ្វើការសិក្សារលំអិត អំពី image បន្ថែមនៅមេរៀនខាងមុខបន្ត html-image chapter



The style Attribute

the style attribute សម្រាប់ ធ្វើការ រចនាបន្ថែមទៅលើ element, ដែលមានដូចជា font, color, ។ល។

ឧទាហរណ៏៖

<!DOCTYPE html>
<html lang="en">
<body>
<p style="color:red">This element using red color</p>
</body>
</html>

The title Attribute

The title attribute កំណត់ពត៌មានបន្ថែមមួយចំនួន សម្រាប់ element

<!DOCTYPE html>
<html lang="km">
<body>
<p title="html-css-java tutorial">Y-C_your class</p>
</body>
</html>
