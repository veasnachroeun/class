CSS Background Shorthand

    The CSS Background Shorthand Property ប្រើដើម្បីកំណត់មុខងារ property ផ្សេងរួមបញ្ជូលគ្នាមកត្រឹមតែមួយបន្ទាត់

ជាឧទាហរណ៏

body {
      background-image: url(/Image/Your\ Class150x150.png);
      background-repeat: no-repeat;
      background-position: right bottom;
      background-attachment: fixed;
       }

អាចសរសេរជា ទម្រង់ Shorthand បែបនេះ

body {
      background: url(/Image/Your\ Class150x150.png)​ no-repeat right bottom fixed;
       }

ចំណាំ៖ ការប្រើ background-shorthand ត្រូវរៀបតាមលំដាប់ តំលៃ Property ដូចខាងក្រោម៖
* background-color
* background-image
* background-repeat
* background-attachment
* background-position
