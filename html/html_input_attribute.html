HTML input attribute



ក្នុង ជំពូកនេះ នឹងធ្វើរការ ពិពណ៌នា អំពីប្រភេទផ្សេងគ្នា រវាង attribute នៃ HTML <input> element.

The value attribute

The input value attribute កំណត់ជាតំលៃ ដំបូង for an input field:

ជាឧទាហរណ៏៖



ប្រើតំលៃដំបូង ទៅលើ input

</form>

The readonly Attribute

The input readonly attribute កំណត់អោយ input field គឺ read-only.

A read-only input field មិនអាចកែបានទេ (តែ, អ្នកប្រើអាចធ្វើការ select, highlight បាន, ហើយក៏អាច copy the text បានផងដែរ).

The value of a read-only input field នឹងធ្វើការផ្ញើចេញ ពេលដែលត្រូវបានបញ្ជូន!

ជាឧទាហរណ៏៖

<form>
     <label for="fname">First name:</label><br>
     <input type="text" id="fname" name="fname" value="veasna"​ readonly><br>
     <label for="lname">Last name:</label><br>
     <input type="text" id="lname" name="lname" value="Chroeun"><br>
     <input type="submit" value="Submit"><br>
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ







The disabled Attribute

The input disabled attribute ធ្វើការកំណត់ទៅលើ input field មិនអោយប្រើបាន.

A disabled input field គឺមិនអាចធ្វើការប្រើបានទាំងអស់ សូម្បីចុច.

តំលៃនៃ disabled input field នឹងមិនត្រូវបានបញ្ជូននោះទេ

ជាឧទាហរណ៏៖

<form>
     <label for="fname">First name:</label><br>
     <input type="text" id="fname" name="fname" value="veasna"​ disabled><br>
     <label for="lname">Last name:</label><br>
     <input type="text" id="lname" name="lname" value="Chroeun"><br>
     <input type="submit" value="Submit"><br>
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ







The size Attribute

The input size attribute កំណត់ជាទំហំ width, របស់ input field.

ទំហំដើមរបស់វា គឺ 20.

ចំណាំ: The size attribute វាអាចបើ្របានតែជាមួយប្រភទ input ដូចជា: text, search, tel, url, email, and password.

ជាឧទាហរណ៏៖

<form>
     <label for="firstname">First Name:</label><br>
     <input type="text" id="firstname" name="firstname"​ size="50"><br>
     <label for="pin">Pin:</label><br>
     <input type="text" id="pin" name="pin" size="4"><br>
     <input type="submit" value="Submit">
     <input type="reset" value="Reset">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ






   


The maxlength Attribute

The input maxlength attribute សម្រាប់កំណត់ចំនួនតួអក្សរដែលអនុញ្ញាតអោយវាយបញ្ចូលក្នុង input field.

ចំណាំ: នៅពេលដែលបានកំណត់ជា maxlength, the input field នឹងមិនអាចវាយបញ្ចូលតួអក្សរលើសពីការកំណត់នោះទេ. ម្យាងទឿត, this attribute មិនបានផ្តល់ជាពត៌មានអោយដឹងនោះទេ. ដូច្នេះ, ប្រសិនបើយើងចង់បញ្ជាក់ទៅកាន់អ្នកប្រើប្រាស់, យើងត្រូវតែសរសេរជា JavaScript code.

ជាឧទាហរណ៏៖

<form>
     <label for="firstname">First Name:</label><br>
     <input type="text" id="firstname" name="firstname"​ size="50"><br>
     <label for="pin">Pin:</label><br>
     <input type="text" id="pin" name="pin" size="4"​ maxlength="4"><br>
     <input type="submit" value="Submit">
     <input type="reset" value="Reset">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ






   

The min and max Attributes

The input min and max attributes សម្រាប់កំណត់តំលៃតិចបំផុត និង ច្រើនបំផុត​ ក្នុង input field.

The min and max attributes ដំណើរការជាមួយប្រភេទ input ដូចជា: number, range, date, datetime-local, month, time and week.​

គន្លឺះ: ការប្រើ the max and min attributes រួមគ្នា គឺដើម្បីជាលក្ខ័ន ទៅតាមលំដាប់នៃតំលៃនិមួយៗ.

ជាឧទាហរណ៏៖
ការកំណត់ max date, a min date, ហើយនិងទៅតាមជួនៃតំលៃ:

<form>
     <label for="datemax">Input date before: 1990-01-01 </label><br>
     <input type="date" id="datemax" name="datemax"​ max="1989-12-31"><br>
     <label for="datemin">Input date after 2000-01-01</label><br>
     <input type="date" id="datemin" name="datemin" min="2000-01-02"​><br>
     <label for="maxandmin">quantity:</label><br>
     <input type="number" id="maxandmin" name="maxandmin" min="1" max="5"​><br>
     <input type="submit" value="Submit">
     <input type="reset" value="Reset">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ










    

The multiple Attribute

The multiple ដំណើរការជាមួយនឹងប្រភេទ input ដូចជា: email, និង file.

<form>
     <label for="files">Plese select Your files to uplaod </label><br>
     <input type="file" id="fils" name="fils" multiple>

     <input type="reset" value="Cancel">

The pattern Attribute

The input pattern សម្រាប់កំណត់តម្លៃជាក់លាក់មួយទៅ the input field នៅពេលដែល form ត្រូវបាន submit.

The pattern attribute ដំណើរការជាមួយប្រភេទ input ដូចជា: text, date, search, url, tel, email, and password.​

គន្លឹះ: ប្រើ the global title attribute បញ្ចាក់ទៅលើ pattern ជាជំនួយដល់អ្នកប្រើ.

ជាឧទាហរណ៏៖

<form>
     <label for="countrycode">Country Code:</label><br>
     <input type="text" id="countrycode" name="countrycode" pattern="[A-za-z]{3}" title="Only three character are allow:">
     <input type="submit" value="Submit">
     <input type="reset" value="Cancel">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ



    

Note: The pattern attribute of the input tag is not supported in Safari 10 (or earlier).


The placeholder Attribute

The input placeholder សម្រាប់បង្ហាញលំនាំជាតំលៃដើម្បីធ្វើការវាយបញ្ចូលក្នុង input field (តាមអ្វីដែលបាន format).

តម្រង់បង្ហាញចេញនៅក្នុង input field មុនពេលអ្នកប្រើធ្វើការបញ្ចូលរាល់តំលៃ.

ជាឧទាហរណ៏៖

<form>
     <label for="phone">Phone Number:</label><br>
     <input type="tel" id="phone" name="phone" placeholder="+855-999-22-7777" pattern="[+]{1}[855]{3}-[0-9]{3}-[0-9]{2}-[0-9]{4}">
     <input type="submit" value="Submit">
     <input type="reset" value="Cancel">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ



Format:+855-###-##-####


    

The required Attribute

The input required attribute សម្រាប់កំណត់ថា input field ត្រូវតែបំពេញមុនពេល submitting the form.

The required attribute ដំណើរការជាមួយប្រភេទ input ដូចជា: text, search, url, tel, email, password, date pickers, number, checkbox, radio, and file.

ជាឧទាហរណ៏៖

<form>
     <label for="birthdate-time">Birthday (date and time):</label><br>
     <input type="datetime-local" id="birthdate-time" name="birthdate-time"​ required>
     <input type="submit" value="Submit">
     <input type="reset" value="Cancel">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ




    

ចំណាំ៖ type="datetime-local" is not supported in Internet Explorer 11 or prior Safari 14.1.


The step Attribute

The input step attribute សម្រាប់កំណត់តំលៃជាចន្លោះៗ​នៅក្នុង input field.

ឧទាហរណ៏: ប្រសិនបើយើងធ្វើការកំណត់ step="2", នោះលេខរបស់យើងនឹងវាយបានដូចនេះ -2, 0, 2, 4,​​​​​ 6 etc.

កន្លឺះ: attribute​ នេះ អាចប្រើរូមជាមួយ max និង min attributes ដើម្បីបង្កើតជាជួរនៃចន្លោះលេខបាន.

The step attribute ដំណើរការជាមួយនឹងប្រភេទ input ដួចជា: number, range, date, datetime-local, month, time and week.

ជាឧទាហរណ៏៖

<form>
     <label for="legalnumber">Intervals legal number:</label><br>
     <input type="number" id="legalnumber" name="legalnumber" step="2">
     <input type="submit" value="Submit">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ


ចំណាំ: កាប្រើ Input restrictions ជាវិធីមួយ, ចំពោះ JavaScript វិញ អាចធ្វើការកំណត់បានច្រើនបន្ថែមទៀតទៅលើ input. ដើម្បីកាន់តែសុវត្ថិភាពទៅលើការកំណត់ក្នុង input, ត្រូវមានកាកំណត់ នៅលើ server ផងដែរ!


The autofocus Attribute

The input​ autofocus attribute សម្រាប់កំណត់ focus ទៅលើ input field រាល់ពេល Page load.

ជាឧទាហរណ៏៖

Let the "First name" input field automatically get focus when the page loads:

<form>
     <label for="fname">Your first name:</label><br>
     <input type="text" id="fname" name="fname" autofocus>
     <label for="lname">Your Last name:</label><br>
     <input type="text" id="lname" name="lname">
     <input type="image" src="../image/button_go.png" alt="Submit" width="45" height="45">
</form>

កូដខាងលើបង្ហាញចេញដូចនេះ








The height and width Attributes

The input height and width attributes សម្រាប់កំណត់ height and width of an input type="image" element.



Tip: ត្រូវតែកំណត់ទាំង height and width attributes ជានិច្ច សម្រាប់កាប្រើ images. រាល់ពេលដែលយើងបាន កំណត់ height and width, ទំហំរបស់ image ត្រូវបានស្គាល់ រាល់ពេលដែល page ធ្វើការ loaded. ផ្ទុយមកវិញបើយើងមិនធ្វើការកំណត់នោះទេ, the browser នឹង មិនស្គាល់ ទំហំរបស់ image នោះទេ, and ហើវាក៏មិនកំនត់នៅទីតាំងជាក់លាក់នោះដែរ វានឹងធ្វើកំណត់ទីតាំងចុចខុសគ្នា រាល់​ page load

ជាឧទាហរណ៏៖

<form>
     <label for="myfile">Select Your File For Upload:</label><br>
     <input list="myfile" id="myfile" name="myfile">
     <input type="image" src="../image/button_go.png" alt="Submit" width="45" height="45">
</form>

The autocomplete Attribute

The input autocomplete attribute សម្រាប់កំណត់ ទៅលើ form ឬ input field បើក autocomplete on or off.

Autocomplete អាចអោយ browser រក្សាទុករាល់តំលៃ នៅពេលអ្នកប្រើធ្វើការវាយ, browser នឹងបង្ហាញចេញនៅមុខងារដែលបានបំពេញពីមុន នៅក្នុង field, ទៅតាមអ្វីដែលបានវាយជាចុងក្រោយ.

The autocomplete attribute ដំណើរការជាមួយ <form> និង <input> types: text, search, url, tel, email, password, datepickers, range, and color.

ជាឧទាហរណ៏៖

An HTML form with autocomplete on, and off for one input field:

<form​ autocomplete="on">
     <label for="fname">First Name:</label><br>
     <input type="text" id="fname" name="fname">
     <label for="lname">Last Name:</label><br>
     <input type="text" id="lname" name="lname">
     <label for="email">Email address:</label><br>
     <input type="email" id="email" name="email" autocomplete="off">
     <input type="submit" value="Submit">
</form>
