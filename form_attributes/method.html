HTML <form> method Attribute



អត្ថន័យនិងការប្រើប្រាស់

The method attribute កំណត់បញ្ជាក់ភាពច្បាស់លាស់ លើ data-form ធ្វើការ ផ្ញើចេញ (the form-data ផ្ញើទៅកាន់ ផេកកំណត់ភាពជាក់លាក់ នៅក្នុង action attribute.

The form-data អាចផ្ញើជា URL variables (with method="get" ឬ​ ជា HTTP post transaction (with method="post"

កំណត់សម្គាល់នៅលើ GET:

  ▀ បន្ថែមពីលើ form-data ទៅក្នុង URL ជាមួយនឹងតម្លៃ name/value
  ▀ ប្រវែងរបស់ URL មានកំណត់ អាចផ្ទុកបានប្រហែល 3000 តួរអក្សរ
  ▀ មិនត្រូវប្រើ GET ចំពោះ DATA ដែលមានសារះសំខាន់ (វាបង្ហាញចេញនៅក្នុង URL)
  ▀ ប្រើច្រើនសំរាប់ form submission កន្លែងដែលអ្នកប្រើប្រាស់ចង់បង្ហាញចេញ Bookmark
  ▀ GET ប្រើសម្រាប់តែ non-secure data, ដូចជា query strings ក្នុង Google

កំណត់សម្គាល់នៅលើ POST:

  ▀ បន្ថែមពីលើ form-data នៅផ្នែកខាងក្នុង តួ របស់ HTTP Request (data មិនបង្ហាញក្នុង URL នោះទេ)
  ▀ មិនមានកំណត់ទំហំ
  ▀ form submission ជាមួយ post គ្មាន Bookmark

Syntax

<form method="get/post">

Attribute Values


Value Description
get Default. Appends the form-data to the URL in name/value pairs: URL?name=value&name=value
post Sends the form-data as an HTTP post transaction

Browser Support

Attribute
autocomplete Yes Yes Yes Yes Yes
