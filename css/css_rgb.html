RGB Color Value

    ក្នុងមេរៀន html ដែលយើងបានសិក្សា កន្លងមក, ចំពោះការប្រើ value ជា RGB មានលក្ខណះសម្គាល់ ដែលតាងជារូបមន្ត ដូចនេះ
rgb(red, green, blue)
ដែលតំលៃ ប៉ារ៉ាមែត្រ មួយៗ របស់ red, green និង blue ជាតំណាងអោយ ពណ៌ ដែលមានតំលៃជាតួលេខ ចាប់ពី 0 ដល់ 255
ដែលមានន័យថាកម្រិតពណ៌មួយៗ គឺមាន 256 x 256 x 256 = 16777216 ជាកម្រិតនៃការប្រែប្រួលរបស់ពណ៌​
     ជា ឧទាហរណ៏ rgb(255, 0, 0) វានឹងបង្ហាញជាពណ៌ក្រហម ពីព្រោះ ពណ៌ក្រហម ត្រូវបានកំណត់តំលៃ ប៉ារ៉ាមែត្រ ខ្ពស់បំផុត គឺ 255 ហើយពីផ្សេងទៀត គឺ green និង blue កំណត់​ត្រឹម 0
      មួយវិញទៀត ប្រសិនបើចង់បាន ពណ៌ខៀវ ត្រូវកំណត់ តំលៃ ប៉ារ៉ាមែត្រ របស់ blue អោយខ្ពស់ បំផុត គឺ 255 និងតម្លៃពីផ្សេងទៀត គឺ red និង green គឺ 0, rgb(0, 0, 255)
      ដើម្បីកំណត់យកពណ៌ខ្មៅ ត្រូវកំណត់តំលៃ ប៉ារ៉ាមែត្រទាំងអស់ ស្មើ 0 ដូចនេះ rgb(0, 0, 0)
      ដើម្បីកំណត់យកពណ៌ស ត្រូវកំណត់តំលៃ ប៉ារ៉ាមែត្រទាំងអស់ ស្មើ 255 ដូចនេះ rgb(255, 255, 255)

RGBA Color Value

rgba color value មុខងារប្រើប្រាស់ដូចទៅនឹង rgb ដែរ គ្រាន់តែបន្ថែម មុខងារមួយទៀត គឺ alpha ដែលមានតួនាទី កំណត់កម្រិតព្រាល ទៅលើពណ៌
syntax:
      rgba(red, green, blue, alpha)
ការកំណត់តំលៃ ប៉ារ៉ាមែត្រ របស់ alpha ចាប់ពី 0.0 ដល់ 1.0
